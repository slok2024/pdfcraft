PDFCraft - 纯浏览器端的 PDF 处理神器

PDFCraft 是一款基于 WebAssembly (WASM) 技术构建的 PDF 处理工具。它的核心特点是 “隐私第一”：所有 PDF 处理（合并、拆分、压缩等）均在你的浏览器本地完成，文件不会被上传到任何服务器。


✨ 核心功能

安全私密：无需上传文件，完全离线化处理，保护敏感隐私。

多语言支持：已针对中文 (zh) 和英文 (en) 进行了精简优化。

多工具合一：支持 PDF 合并、拆分、页面提取、压缩及格式转换。

极速性能：利用 PyMuPDF 与 QPDF 的 WASM 强力驱动，处理速度媲美原生应用。


使用node.js v24.13.0编译好的文件，并删减了无用的多国语言包。可以使用CF的pages直接上传这个压缩包部署使用，中间提示文件大小不用管。
