# PDFCraft 🚀  
**纯浏览器端的 PDF 处理神器（隐私优先）**

PDFCraft 是一款基于 **WebAssembly (WASM)** 构建的现代化 PDF 处理工具。  
秉承 **“隐私第一”** 的设计理念，所有 PDF 操作 **完全在浏览器本地完成**，文件不会被上传到任何服务器。

> ✅ 无后端  
> ✅ 无数据上传  
> ✅ 即开即用

---

## ✨ 核心特性

- 🔐 **安全私密**
  - 所有 PDF 处理均在本地浏览器完成
  - 完全离线化，不经过任何服务器
  - 特别适合处理敏感或私密文档

- 🌍 **多语言支持**
  - 精简优化的语言包
  - 当前支持：
    - 中文（`zh`）
    - 英文（`en`）

- 🧰 **多工具合一**
  - PDF 合并
  - PDF 拆分
  - 页面提取
  - PDF 压缩
  - PDF 格式转换

- ⚡ **极速性能**
  - 基于 **PyMuPDF** 与 **QPDF** 的 WASM 构建
  - 性能接近原生桌面应用
  - 大文件处理依然流畅

---

## 🛠 技术栈

- **WebAssembly (WASM)**
- **PyMuPDF (WASM)**
- **QPDF (WASM)**
- **Node.js v24.13.0**

## 📦 构建说明

- 使用 **Node.js v24.13.0** 编译生成
- 已移除无用的多国语言包，显著减小体积
- 更适合静态托管与边缘部署

---

## ☁️ 部署方式（Cloudflare Pages）

PDFCraft 可直接部署在 **Cloudflare Pages** 上：

1. 将已编译好的 **压缩包** 上传至 CF Pages
2. 构建提示文件体积过大时 **可直接忽略**
3. 部署完成后即可访问，无需任何后端配置

> 💡 非常适合个人工具站、内网使用或隐私优先场景

- 隐私敏感文档操作

- 替代在线 PDF 上传网站
